# 🍔 Portal Burger - Order Management System

## 🚀 Project Overview
Portal Burger is a comprehensive web application designed to streamline order management for a burger restaurant. It allows customers to browse the menu and place orders intuitively, 
while providing an administrative dashboard to manage inventory, process sales, and control the business workflow in real-time.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript 
- **Backend:** PHP
- **Database:** MySQL
- **Hosting/Server:** InfinityFree (Apache)
- **Styling:** Bootstrap / Custom CSS

## ⚙️ Features
- **Dynamic Catalog:** Product display with updated categories and pricing.
- **Shopping Cart System:** Interactive system to add, edit, and remove items before confirming an order.
- **Admin Dashboard:** Full control over inventory (Product CRUD).
- **Sales Management:** Real-time tracking of orders and preparation status.
- **Responsive Design:** Fully optimized for mobile and desktop devices.

## ▶️ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GermanLL/Portalburger.germanluna.git

2. **Set up the environment:**

- You will need a local server like XAMPP.

- And also the HeidiSQL application.
  
- Open the project in Visual Code and run "composer install" in the console.

3. **Database Setup:**

- Access HeidiSQL application.

- Import the u188377931_burger.sql file located in the project's root folder.


4. **Connection Config:**

- Update the .env files, add only DB_DATABASE and DB_USERNAME.


5. **Launch:**
- Once everything is done, type the following in the terminal: php artisan serv
- Open your browser and go to http://127.0.0.1:8000/
